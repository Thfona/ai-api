# IARA

## Scripts

```bash
# Install dependencies
$ npm install

# Start application
$ npm start
$ npm run start:dev
$ npm run start:debug
$ npm run start:prod

# Build
npm run build

# Lint
$ npm run lint
$ npm run lint:fix

# Test
$ npm run test
$ npm run test:watch
$ npm run test:cov
$ npm run test:debug
$ npm run test:e2e
```
